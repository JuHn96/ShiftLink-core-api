# 🛠️ ShiftLink 통합 기술 명세서 (Technical Specification)

이 문서는 ShiftLink 프로젝트의 시스템 구조와 핵심 기술 스펙을 정리하는 목차 레이더입니다.

* 상세 **소프트웨어 아키텍처 명세**는 `architecture.md`를 참조하세요.
* 핵심 코어 **PostgreSQL 데이터베이스(DB) 테이블 명세**는 `db_spec_postgres.md`를 참조하세요.
* 유지보수 및 속도 최적화를 위한 **아키텍처 의사결정 이력(ADR)**은 `dev_knowledge_adrs.md`를 참조하세요.

---

*(이 문서는 향후 API 엔드포인트 세팅 및 마이크로서비스 간 통신 프로토콜 명세가 발생할 때 구체적으로 채워나갈 예정입니다. 현재는 DB 모델링 마일스톤 단계이므로 참조 문서 링크만 제공합니다.)*
